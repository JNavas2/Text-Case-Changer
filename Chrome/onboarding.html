Welcome to Text Case Changer

Extension to quickly change the case of selected editable text.
Use the context menu to select the case transformation you want:

  • lower case
  • UPPER CASE
  • Invert cASE
  • Sentence case
  • Start Case
  • Title Case (AP style*)
  • camelCase
  • snake_case

* All words in titles are capitalized except for certain short words,
such as articles and short prepositions. 

NOTE: Words in all caps (e.g., NASA) are not changed.

PRIVACY: This extension does not collect user information.
