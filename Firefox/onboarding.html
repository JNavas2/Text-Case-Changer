Welcome to Text Case Changer

Extension to quickly change the case of selected editable text.
Use context menu or shortcut to select desired case change:

  • lower case
  • UPPER CASE
  • Invert cASE
  • Sentence Case.
  • Start Case
  • Title Case (AP style)
  • camelCase
  • snake_case

NOTES:

  1. Shortcuts can be set for case functions in the browser.
  2. Words in all caps (e.g., NASA) are not changed.
  3. All words in AP style titles are capitalized except for certain
    short words, such as articles and short prepositions.

PRIVACY: This extension does not collect user information.
