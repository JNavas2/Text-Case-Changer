# Text Case Changer
To change the case of editable text, select the text, 
and then invoke a context menu to choose the desired case: 
* lower case 
* UPPER CASE 
* Invert cASE 
* Sentence case 
* Start Case 
* Title Case 
* camelCase 
* snake_case

PRIVACY: This extension does <b>not</b> collect user information.
